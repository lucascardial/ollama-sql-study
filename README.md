# IA Local Text-to-SQL com Ollama + PostgreSQL

Projeto acadêmico demonstrando a conversão de linguagem natural em consultas SQL reais utilizando um modelo de IA executado localmente com Ollama.

A aplicação interpreta perguntas em linguagem natural, gera queries PostgreSQL automaticamente, executa as consultas em um banco de dados real e retorna os resultados ao usuário.

## Stack

- Ollama
- Llama 3
- Node.js + TypeScript
- PostgreSQL
- Docker Compose

## Recursos

- Execução 100% local
- Conversão de linguagem natural em SQL
- Geração automática de queries PostgreSQL
- Validação de queries somente leitura
- Execução real no banco de dados
- Respostas em linguagem natural
- Carregamento dinâmico do schema SQL
- Terminal com saída colorida

## Objetivo

Demonstrar de forma prática como modelos de linguagem locais podem interagir com bancos relacionais através da técnica de Text-to-SQL, sem dependência de APIs externas.
